# FDE-desktop
FDE is a desktop environment based on ULDE.

It uses:
- xcompmgr
- ULDE components (tint2, openbox, etc.)
- UX apps (Firefox, Vim, Wallpaper)
- Customized Tint2 panel

# How to install:
Option 1:
1. In terminal, type "sudo apt install lightdm" - this makes logging into FDE easier.
2. Then, type "git clone https://github.com/Xelron161/FDE-desktop" - type in the actual release tag you want here.
3. Next, type in "sudo dpkg -i PATH/TO/FDE.deb" - use the actual path to the FDE .deb file.
4. If you have all depends, that's good. Else, type in "sudo apt --fix-broken install" and do step 4 again.
5. Reboot, choose FDE, login, Done! Enjoy FDE ;)

Run FDE Installer.
